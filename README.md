## CAMPUSS+微信小程序

### 简介

校园互助平台 

### 技术栈：

SpringBoot+Mysql+Druid+微信原生 JS+weUI 组件库

### 功能

跳蚤市场、跑腿代拿、拼车滴滴、学习心得交流和展示学校官方新闻数据

### 负责模块：

**后端开发和部署**

- 数据库设计和业务逻辑分析
- 服务端携带小程序唯一标识和开发者密钥等参数请求微信服务器获取 oppenid 并保存到数据库，标识用户是否授权过小程序
- 完成所有业务功能的代码编写
- 调用微信官方消息通知 API，实现用户订单状态的自动推送
- 部署采用 HTTPS 协议，实现和微信小程序前端接轨，完成上线
